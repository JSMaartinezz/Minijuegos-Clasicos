### Título del proyecto: Adivina el número.

## Cómo jugar:
- El navegador proyecta un número que no se puede ver.
- El mismo navegador te dirá pista de más alto o más bajo para acercarte poco a poco.
- Los números irán del 1 al 100 inclusive.
- Se contarán los intentos que has realizado hasta adivinarlo.
- Tendrás 30 segundos para adivinarlo.

## Tecnologías usadas:
- HTML
- CSS
- JavaScript
