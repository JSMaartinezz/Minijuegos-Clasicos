### PIEDRA, PAPEL O TIJERAS.

## EXPLICACIÓN SENCILLA DEL PROYECTO.
- El proyecto consistirá en recrear el famoso juego de piedra, papel o tijeras donde competirás contra la máquina.
- Ganará el primero que llegue a 5.
- La novedad es que habrá una serie de "potenciadores" que tanto tú como la máquina podréis usar.

## TECNOLOGÍAS USADAS.
- HTML
- CSS
- JavaScript
